Cars Users

Hello! Welcome to the system that consumes an API related to cars and users.
This is a system developed in Angular and Spring Boot.
The main objective of the project is to cover the concepts of SPAs, Authentication and Authorization with JWT Tokens, and much more. The source code of the project can be accessed by clicking on the Github link in the side menu.

Feel free to test the system and leave your suggestion.
